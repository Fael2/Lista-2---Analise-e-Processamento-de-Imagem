##  Descrição Geral
Este repositório contém as implementações e análises referentes à **Lista 2**, abordando:

1. **Questão 1 — Comparação entre algoritmos de segmentação de imagens (Otsu e K-means)**
2. **Questão 2 — Representação geométrica de regiões (Fecho Convexo)**

Os experimentos foram desenvolvidos em **Python 3.11** com uso das bibliotecas **OpenCV**, **NumPy** e **Matplotlib**.  
O relatório final (em PDF) descreve a fundamentação teórica, metodologia, resultados e conclusões de ambas as etapas.

# Instalar dependências
pip install -r requirements.txt 

##  Executar Questão 1 - Segmentação
Dentro da pasta **src**, execute no terminal:

##  Executar Questão 2 - Representação
Dentro da pasta **src**, execute no terminal:
